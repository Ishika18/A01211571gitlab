## What is .md extendion?
**.md extension** stands for *Markdown*, which is a *markup language* with plain text formatting syntax. It is a language used to create readme files for github.
## Why is .md extension preferred over .txt ot .html
github prefers markdowm over .txt or .html because
1. it is easy to read
2. it is plain text so it is source controlled easily
